# xeno.video
##  Getting started

To configure, copy `.env.dist` to `.env`, and modify that copy.

### Wercker it

* Get [wercker](http://wercker.com/downloads/)
* `wercker build`

### Local development

* npm install

* Run `gulp build`
* Run `./bin/www`
* Visit, by default, http://127.0.0.1:3000/
